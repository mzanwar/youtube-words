Getting started:

1. Make sure you have pip installed
2. `python3 - venv venv`
3. `source venv/bin/activate` 
4. run a `pip install -r requirements.txt`
5. Get redis and ES going
6. Get brew
7. brew install redis
8. brew install elasticsearch
9. redis-server
10. elasticsearch

11. ./start.sh coronavirus 1
